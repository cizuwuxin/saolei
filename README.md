# 扫雷

一个用 Material Design 3 风格写的扫雷游戏，单文件 HTML，零依赖。

## 🎮 试玩

👉 **[点我开始游戏](https://cizuwuxin.github.io/saolei/saolei.html)**

手机、平板、电脑都行，自动跟随系统的浅色/深色主题，也可以在游戏内手动切换。

## 操作

| 操作 | 鼠标 | 触屏 | 键盘 |
|---|---|---|---|
| 翻开格子 | 左键 | 轻点 | 方向键 + Enter |
| 标记雷 | 右键 | 长按 | F 键 |
| 数字展开 | 双击 | — | 双击 |
| 撤销 | 顶栏 | 顶栏 | Ctrl+Z |
| 新游戏 | 顶栏 | 顶栏 | R 键 |

## 特点

- Material Design 3 视觉系统
- 传统 / 无猜双模式
- 自定义难度（行列/雷数）
- 主题、问号标记、音效反馈都可调
- 首次点击保护
- 撤销 / 重开
- 响应式布局，手机友好

## 部署

直接双击 `saolei.html` 即可在本地运行。也可部署到任何静态托管（如 GitHub Pages）。
