# 扫雷 / Minesweeper

单文件 HTML 扫雷游戏，使用 Material Design 3 视觉风格，无外部依赖。

A single-file HTML Minesweeper game using Material Design 3, with no external dependencies.

## 试玩 / Play

👉 https://cizuwuxin.github.io/minesweeper/

手机、平板、电脑都能玩，自动跟随系统的浅色/深色主题，也可以在游戏内手动切换。

Works on phones, tablets, and desktops. Follows system light/dark theme, with a manual toggle in the game.

## 操作 / Controls

| 操作 Action | 鼠标 Mouse | 触屏 Touch | 键盘 Keyboard |
|---|---|---|---|
| 翻开格子 Reveal | 左键 Left | 轻点 Tap | 方向键 + Enter |
| 标记雷 Flag | 右键 Right | 长按 Long press | F |
| 数字展开 Chord | 双击 Double | — | 双击 Double |
| 撤销 Undo | 顶栏 Toolbar | 顶栏 Toolbar | Ctrl+Z |
| 新游戏 New | 顶栏 Toolbar | 顶栏 Toolbar | R |

## 特点 / Features

- Material Design 3 视觉系统 / Material Design 3 visuals
- 传统 / 无猜双模式 / Classic and no-guess dual modes
- 自定义难度（行列/雷数）/ Custom difficulty (rows / cols / mines)
- 主题、问号标记、音效反馈都可调 / Theme, question marks, sound effects
- 首次点击保护 / First-click safety
- 撤销 / 重开 / Undo and restart
- 响应式布局，手机友好 / Responsive layout, mobile-friendly

## 部署 / Deployment

直接双击 `index.html` 即可在本地运行。也可部署到任何静态托管（如 GitHub Pages）。

Just open `index.html` locally, or deploy to any static host (e.g. GitHub Pages).
